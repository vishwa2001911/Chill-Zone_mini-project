# Movie Ranking Website
This is a university project designed to create a movie ranking website using the TMDB API.

## Features
* View movie / Tv-series ranks
* Vive Tv-series information
* View movie information

## Prerequisites
* TMDB API Key
* JavaScript
* Vue js
